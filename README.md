# نظام سوبر آبل — دليل الرفع على استضافتك (cPanel)

هذا الإصدار مبني بـ **PHP + MySQL** (مو Node.js)، وهي التقنية اللي كل استضافات cPanel تقريبًا تدعمها جاهزة بدون أي تركيب إضافي.

تمّ اختبار كل الوظائف فعليًا (تسجيل دخول، إضافة موظفين، مهام مشتركة، دوام، نقاط) وتعمل بشكل صحيح ✅

---

## الخطوة 1: إنشاء قاعدة البيانات

1. ادخل على **cPanel** → دور لأيقونة **MySQL Databases** (قواعد بيانات MySQL)
2. تحت "Create New Database" اكتب اسم زي `superapple` واضغط Create
   - رح يصير الاسم الكامل تلقائيًا شي زي: `yourcpanel_superapple`
3. تحت "MySQL Users" أنشئ مستخدم جديد بكلمة سر قوية
4. تحت "Add User to Database" اربط المستخدم بالقاعدة وأعطه **All Privileges**
5. **احتفظ بثلاث معلومات:** اسم القاعدة الكامل، اسم المستخدم الكامل، كلمة السر

## الخطوة 2: استيراد جدول البيانات

1. من cPanel افتح **phpMyAdmin**
2. اختر قاعدة البيانات اللي أنشأتها من القائمة الجانبية
3. اضغط تبويب **Import** (استيراد)
4. اختر ملف `schema.sql` المرفق وارفعه، ثم اضغط **Go**
5. رح تظهر 7 جداول (users, tasks, task_assignees, attendance, points, notifications, settings)

## الخطوة 3: تعديل بيانات الاتصال

افتح ملف `api/config.php` بأي محرر نصوص (أو من File Manager بالـ cPanel مباشرة → Edit) وعدّل:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'yourcpanel_superapple');   // اسم القاعدة اللي أنشأتها
define('DB_USER', 'yourcpanel_dbuser');       // اسم المستخدم اللي أنشأته
define('DB_PASS', 'كلمة السر تبعت المستخدم');
```

## الخطوة 4: رفع الملفات

عن طريق **File Manager** بالـ cPanel أو برنامج FTP (مثل FileZilla):

1. ادخل لمجلد `public_html` (أو مجلد فرعي إذا بدك النظام على رابط فرعي، مثل `public_html/team`)
2. ارفع كل هاي الملفات محافظًا على نفس الهيكلية:
   ```
   index.html
   api/
     ├── index.php
     ├── config.php   (بعد ما عدّلته بالخطوة 3)
     └── db.php
   ```
   (ملف `schema.sql` لا يُرفع، هو فقط للاستيراد بالخطوة 2)

## الخطوة 5: افتح الموقع

روح لرابط موقعك (مثال: `https://yourdomain.com/` أو `https://yourdomain.com/team/`)

أول مرة رح تظهر شاشة "إعداد النظام لأول مرة" — أنشئ فيها حساب المدير الرئيسي (اسمك + إيميل + كلمة سر)، وبعدها جاهز تضيف موظفينك وتبلش تستخدم النظام. كل موظف بياخد رابط الموقع نفسه ويسجل دخول بالحساب اللي تنشئه له من لوحة "الموظفون".

---

## ملاحظات مهمة

- ✅ كلمات السر الآن **مشفّرة بالكامل** بقاعدة البيانات (bcrypt) — أأمن بكثير من النسخة التجريبية السابقة.
- ✅ البيانات محفوظة بقاعدة بيانات حقيقية خاصة فيك، مو مشتركة مع أي حد غير فريقك.
- ⚠️ لو ظهرت رسالة "تعذر الاتصال بقاعدة البيانات" عند فتح الموقع، رجّع تأكد من بيانات `api/config.php` (الخطوة 3).
- 📱 ميزة إشعارات الواتساب **غير مفعّلة حاليًا** (اتفقنا نأجّلها). النظام جاهز لها لاحقًا — بس تبني سيرفر واتساب منفصل وتربطه من تبويب "ربط واتساب" بلوحة المدير.
- استخدم HTTPS إذا متوفر عندك (SSL) — أغلب استضافات cPanel توفره مجانًا (Let's Encrypt) من نفس اللوحة، وهاد بيخلي تسجيل الدخول أأمن.
